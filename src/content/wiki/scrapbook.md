# Scrapbook

Scrapbook is Hack Club's daily update feed at [scrapbook.hackclub.com](https://scrapbook.hackclub.com). Every day, hundreds of Hack Clubbers post what they're working on.

## How It Works

1. Post an update in the `#scrapbook` channel on [[Slack]]
2. It automatically appears on your public Scrapbook profile
3. Add images, videos, or links to show what you're building
4. Build a streak by posting every day

## Your Profile

Every Hack Clubber gets a public profile page at `scrapbook.hackclub.com/username`. Your profile shows:

- All your updates in reverse chronological order
- Your current streak
- A CSS customization if you've set one up

## CSS Customization

One of the most fun parts of Scrapbook: you can customize your profile with custom CSS. Post your CSS in a `#scrapbook-css` channel update and it'll be applied to your profile.

Some Hack Clubbers have made incredibly creative profiles — animated backgrounds, custom fonts, pixel art.

## Streaks

Keep a streak by posting every day. The Scrapbook bot tracks your streak and announces milestones in `#scrapbook`.

## What to Post

Anything you're working on:

- A screenshot of code you wrote
- A demo of something you built
- A photo of hardware you're wiring
- Notes from something you learned
- A bug you fixed (or are struggling with)

No post is too small. The goal is consistency and sharing, not perfection.

## The API

Scrapbook has a public API at `scrapbook.hackclub.com/api`. You can use it to embed your updates in other projects or build tools around it.
